Class InvalidatedWeakEventHandler<TTarget>

A lightweight proxy instance that will subscribe to a given event with a weak reference to the subscribed target. If the subscriber is garbage collected, then only this WeakEventHandler will remain subscribed and keeped in memory instead of the actual subscriber. This could be considered as an acceptable solution in most cases.

public void Dispose()
Remarks

Call Dispose() when you are finished using the InvalidatedWeakEventHandler<TTarget>. The Dispose() method leaves the InvalidatedWeakEventHandler<TTarget> in an unusable state. After calling Dispose(), you must release all references to the InvalidatedWeakEventHandler<TTarget> so the garbage collector can reclaim the memory that the InvalidatedWeakEventHandler<TTarget> was occupying.
